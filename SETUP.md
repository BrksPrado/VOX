# SETUP — VOX

---

## Instalar as dependências

Dentro da pasta do projeto, rodar:

```bash
npm install
```

Vai baixar tudo que o projeto precisa, ai espera terminar.

---

## Rodar o projeto

```bash
npm run dev
```

Abrir o navegador em: http://localhost:5173

---

## Outros comandos

```bash
npm run build    # gera a versão de produção
npm run preview  # visualiza o build gerado
```
